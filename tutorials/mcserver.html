    • Minecraft-Server mit ngrok erstellen

    Benötigt:

    • Java
    • server.jar
    • Ngrok-Account
    • ngrok.exe
    • run.bat

    1. Java

    [Optional] Wenn du Java noch nicht auf deinem Computer installiert hast,

    lade es dir auf https://www.java.com/de/download/ runter.

    2. server.jar

    Lade dir im Minecraft-Launcher die server.jar runter:

    3. Ngrok-Account

    Erstelle dir einen Account bei https://www.ngrok.com

    Klicke nach dem Anmelden im Dashboard auf "Download for Windows" und folge der Installationsanleitung.

    4. run.bat

    Um den Server vernünftig zu starten, lade dir die run.bat-Datei runter und packe sie in den gleichen Ordner, wie die server.jar.

    Wenn du die run.bat mit einem Text-Editor öffnest, kannst du sie ein bisschen anpasen:

    java -Xmx1024M -Xms1024M -jar server.jar

    Wobei Xmx der maximal nutzbarer Arbeitsspeicher für den Server ist.
