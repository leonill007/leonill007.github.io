Minecraft-Server mit ngrok erstellen:

Benötigt:

Java

server.jar

Ngrok-Account

ngrok.exe

run.bat

1. Java

[Optional] Lade dir Java auf https://www.java.com/de/download/ runter.

2. server.jar

Lade dir im Minecraft-Launcher die server.jar runter:

3. Ngrok-Account

Erstelle dir einen Account bei https://www.ngrok.com

Klicke nach dem Anmelden im Dashboard auf "Download for Windows" und folge der Installationsanleitung.

4. run.bat
